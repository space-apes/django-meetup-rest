Welcome to Meetup Street, anonymous!

  • This is a Django Rest Framework based RESTful api for a meetup type social media webapplication.
  • Endpoints serve and accept JSON data.
  • The root url is a placeholder for a front-end endpoint.
  • Authentication is meant to be handled through JSON Web Tokens. However, basic username/password authentication can also be used for demo purposes.
  • See the README.md for credentials!

to access the api, try any of the following:

  • /api/ login or see other endpoints here
  • /api/users/
    • superuser: {C,R}
    • anonymous: {C}
  • /api/users/<user_pk>/
    • superuser: {R,U,D}
    • user.pk == pk: {R,U,D}
  • /api/meetup_groups/
    • superuser: {C,R}
    • authenticated user: {C}
    • any user: {R}
  • /api/meetup_groups/<meetup_pk>/
    • superuser: {R,U,D}
    • admin of meetup_group: {R,U,D}
    • any user: {R}
  • /api/events/
    • superuser: {C,R}
    • any admin of a meetup_group: {C,R}
    • any user: {R}
  • /api/events/<event_pk>/
    • superuser: {R,U,D}
    • the host of the event: {R,U,D}
    • any user: {R}
  • /api/tags/
    • superuser: {C,R}
    • any authenticated user: {C,R}
    • any user: {R}
  • /api/tags/<tag_pk>/
    • superuser: {U,D}
    • any user: {R}
  • /api/users/<user_pk>/meetup_groups/ meetup_groups filtered by user membership
    • user.id == user_pk: {R}
  • /api/meetup_groups/<meetup_group_pk>/events/ events filtered by association with meetup_group
    • users who are members of meetup group: {R}
